# Calculadora 2025

Trata-se de uma aplicação com funcionalidades de uma calculadora básica contendo algumas funções a mais, como por exemplo, a tabuada de um número, e o histórico de operações!

O programa pode receber dois valores numéricos para realizar operações de soma, subtração, multiplicação e divisão, e há validação pro caso de digitar operação inválida. Pode receber somente 1 valor caso escolha a opção de  tabuada. Também existe a opção de sair, que fecha o app.

## Uso
Basta digitar o número correspondente da operação a ser realizada,
```
1 - Somar
2 - Subtrair
3 - Multiplicar
4 - Dividir
5 - Tabuada
6 - Histórico de operações
S - Sair
```
e em seguida digitar os números para a operação para obter o resultado.
```
........................
Digite o primeiro número:

|

........................
Digite o segundo número:

|
```

#### Obs: Quando precisar usar números com casas decimais utilize ","(vírgula) e não "."(ponto)!
ex:
```
........................
Digite o primeiro número:

54,23
```
